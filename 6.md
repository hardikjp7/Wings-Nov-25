### **Dashboard Size** â­ **EXAM IMPORTANT!**

**Dashboard â†’ Size options:**

### **1. Fixed Size**
- Exact pixel dimensions
- Example: 1200 Ã— 800
- No responsive behavior

**When to use:**
- Specific display requirements
- Kiosk mode
- Print layout

---

### **2. Automatic**
- Fills browser window
- Scales to fit
- Responsive to window resize

**When to use:**
- Web display
- Unknown screen sizes

- Min and max dimensions
- Scales within bounds
- Example: 800-1400 wide

**When to use:**
- Responsive with limits
- Avoid too small/large
- Best of fixed + automatic
