# Module 2: Building Basic Visualizations - Detailed Hands-On Notes

## 1. Understanding the Tableau Workspace

### Key Components:

**A. Data Pane (Left Side)**
- **Dimensions:** Qualitative fields (blue color) - Category, Region, Product Name, Date parts
- **Measures:** Quantitative fields (green color) - Sales, Profit, Quantity
- **Sets:** Custom groupings
- **Parameters:** User-defined variables

**B. Shelves Area (Top)**
- **Columns Shelf:** Defines horizontal axis
- **Rows Shelf:** Defines vertical axis
- **Filters Shelf:** Filters data shown in view
- **Marks Card:** Controls visual properties
- **Pages Shelf:** Creates animated views

**C. Marks Card**
- **Type Dropdown:** Automatic, Bar, Line, Area, Square, Circle, Shape, Text, Map, Pie, Gantt Bar, Polygon, Density
- **Color:** Changes color of marks
- **Size:** Changes size of marks
- **Label:** Shows text on marks
- **Detail:** Adds granularity without visual encoding
- **Tooltip:** Information on hover

**D. View/Canvas (Center)**
- Where visualization appears
- Drag and drop fields here

---

## 2. Creating Basic Charts

### A. Bar Chart (Vertical)

**Steps:**
1. Drag **Dimension** to Columns shelf (e.g., Category)
2. Drag **Measure** to Rows shelf (e.g., Sales)
3. Tableau creates vertical bars automatically

**Hands-On Example:**
```
Data: Sample-Superstore
Task: Show Sales by Category

Steps:
1. Drag "Category" to Columns
2. Drag "Sales" to Rows
3. Result: 3 bars (Furniture, Office Supplies, Technology)
```

**Customization:**
- **Sort bars:** Click sort icon on toolbar (ascending/descending)
- **Change color:** Drag field to Color on Marks card
- **Add labels:** Drag "Sales" to Label on Marks card
- **Change bar color:** Click Color → Select color

---

### B. Bar Chart (Horizontal)

**Steps:**
1. Drag **Dimension** to Rows shelf (e.g., Sub-Category)
2. Drag **Measure** to Columns shelf (e.g., Profit)

**Hands-On Example:**
```
Task: Show Profit by Sub-Category (horizontal)

Steps:
1. Drag "Sub-Category" to Rows
2. Drag "Profit" to Columns
3. Result: Horizontal bars showing profit for each sub-category
```

**Quick Switch:**
- To convert vertical to horizontal: Click **Swap Rows and Columns** icon (toolbar)
- Keyboard shortcut: **Ctrl + W**

---

### C. Stacked Bar Chart

**Steps:**
1. Create basic bar chart
2. Drag another **Dimension** to Color on Marks card

**Hands-On Example:**
```
Task: Show Sales by Region, colored by Category

Steps:
1. Drag "Region" to Columns
2. Drag "Sales" to Rows
3. Drag "Category" to Color (Marks card)
4. Result: Stacked bars showing category breakdown per region
```

**Side-by-Side Bars (Grouped):**
- Drag the dimension from Color to Columns shelf (before or after existing dimension)

---

### D. Line Chart

**Steps:**
1. Drag **Date** dimension to Columns
2. Drag **Measure** to Rows
3. Tableau defaults to line chart for dates

**Hands-On Example:**
```
Task: Show Sales trend over time

Steps:
1. Drag "Order Date" to Columns
2. Drag "Sales" to Rows
3. Result: Line showing sales by year (default aggregation)
```

**Date Granularity:**
- Click **+ icon** on Order Date pill: Drill down (Year → Quarter → Month → Day)
- Click **- icon**: Drill up
- Right-click date pill → Change date level:
  - Year
  - Quarter
  - Month
  - Week
  - Day
  - Exact Date

**Multiple Lines:**
- Drag dimension to Color on Marks card

**Example:**
```
Task: Compare sales trends by Region

Steps:
1. Order Date on Columns (by Month)
2. Sales on Rows
3. Region on Color
4. Result: 4 lines (one per region)
```

---

### E. Area Chart

**Steps:**
1. Create line chart
2. Change mark type to **Area** from Marks card dropdown

**Or:**
1. Drag Date to Columns
2. Drag Measure to Rows  
3. Click **Marks dropdown** → Select **Area**

**Hands-On Example:**
```
Task: Show cumulative sales over time

Steps:
1. Order Date (Month) on Columns
2. Sales on Rows
3. Change mark type to Area
4. Drag "Category" to Color for stacked area chart
```

**Stacked vs Individual Areas:**
- **Stacked:** Default when dimension on Color
- **Individual:** Drag dimension to Rows shelf instead

---

### F. Pie Chart

**Steps:**
1. Drag **Measure** to Columns or Rows
2. Change mark type to **Pie**
3. Drag **Dimension** to Color
4. Drag **Measure** to Angle (automatically done)

**Hands-On Example:**
```
Task: Show Sales distribution by Category

Steps:
1. Drag "Sales" to Columns
2. Change mark type to Pie
3. Drag "Category" to Color
4. Drag "Category" to Label (show category names)
5. Drag "Sales" to Label (show values)
```

**Formatting:**
- Right-click label → **Format** → Adjust font, alignment
- Click Color → **Edit Colors** → Assign specific colors

**Exam Tip:** Pie charts best for showing parts of whole (3-5 categories max)

---

### G. Scatter Plot

**Steps:**
1. Drag **Measure 1** to Columns
2. Drag **Measure 2** to Rows
3. Drag **Dimension** to Detail (one mark per item)
4. Change mark type to **Circle** (if not automatic)

**Hands-On Example:**
```
Task: Show relationship between Sales and Profit

Steps:
1. Drag "Sales" to Columns
2. Drag "Profit" to Rows
3. Drag "Product Name" to Detail
4. Result: Each product as a dot
```

**Enhanced Scatter Plot:**
```
Add more insights:
- Drag "Category" to Color (color by category)
- Drag "Quantity" to Size (size by quantity)
- Drag "Product Name" to Label (show names)
```

---

### H. Text Table (Crosstab)

**Steps:**
1. Drag **Dimension(s)** to Rows and Columns
2. Drag **Measure** to Text on Marks card
3. Mark type automatically changes to **Text**

**Hands-On Example:**
```
Task: Create sales table by Region and Category

Steps:
1. Drag "Region" to Rows
2. Drag "Category" to Columns
3. Drag "Sales" to Text
4. Result: Table with sales values in cells
```

**Formatting:**
- Click Text on Marks → **Format** → Adjust alignment, font
- Right-click measure → **Format** → Number format (currency, percentage, etc.)

## 3. Show Me Panel

**What it is:** Smart chart recommendation panel based on selected fields

**Location:** Top-right corner of workspace (grid icon with 24 chart types)

**How to use:**
1. Select fields from Data pane (Ctrl+Click for multiple)
2. Click **Show Me** button
3. Available chart types highlighted
4. Unavailable charts grayed out
5. Click desired chart type

**Hands-On Practice:**
```
Experiment with different field combinations:

1. Select "Category" → Show Me highlights: Bar, Text Table, Treemap, etc.
2. Select "Category" + "Sales" → Show Me highlights: Bar, Line, Pie, etc.
3. Select "Order Date" + "Sales" → Show Me highlights: Line, Area, etc.
4. Select "Sales" + "Profit" → Show Me highlights: Scatter plot, etc.
```

**Chart Requirements (Key ones for exam):**

| Chart Type | Requirements |
|-----------|-------------|
| Bar Chart | 0-1 Dimension, 1-2 Measures |
| Line Chart | 1+ Dimensions (one must be date), 1-2 Measures |
| Pie Chart | 1-2 Dimensions, 1-2 Measures |
| Scatter Plot | 2-4 Measures, 0-1 Dimension |
| Text Table | 1+ Dimensions, 1+ Measures |
| Heat Map | 1+ Dimensions, 1-2 Measures |
| Treemap | 1+ Dimensions, 1-2 Measures |

**Exam Tip:** Show Me panel shows minimum requirements. Hover over chart type to see requirements.

## 5. Filters

### A. Dimension Filters

**Types:**
1. **General** - Select from list
2. **Wildcard** - Match text pattern
3. **Condition** - Filter by formula
4. **Top** - Top/Bottom N items

---

**Method 1: Drag to Filters Shelf**
```
Task: Show only Furniture and Technology categories

Steps:
1. Create visualization
2. Drag "Category" to Filters shelf
3. Filter dialog appears
4. Select "Furniture" and "Technology"
5. Click OK
```

**Method 2: Right-click Dimension**
```
1. Right-click "Category" in Data pane
2. Select "Show Filter"
3. Filter card appears on right
4. Check/uncheck categories
```

---

**General Filter (Checklist):**
- Default type for dimensions
- Shows all values
- Select/deselect items
- **All** checkbox to select/deselect all
- **Exclude** checkbox to reverse selection

**Hands-On Example:**
```
Task: Show only 3 specific regions

Steps:
1. Drag "Region" to Filters
2. Select: East, West, South
3. Click OK
```

---

**Wildcard Filter:**
```
Task: Show only products containing "Chair"

Steps:
1. Drag "Product Name" to Filters
2. Click "Wildcard" tab
3. Select "Contains"
4. Type "Chair"
5. Click OK
```

**Wildcard Options:**
- **Contains:** Value contains text
- **Starts with:** Value begins with text
- **Ends with:** Value ends with text
- **Exactly matches:** Exact match

---

**Condition Filter:**
```
Task: Show only categories with sales > 500,000

Steps:
1. Drag "Category" to Filters
2. Click "Condition" tab
3. Select "By field"
4. Field: Sales
5. Aggregation: Sum
6. Condition: > 500000
7. Click OK
```

**Condition Options:**
- Compare by field (measure)
- Use formula: Write custom calculation
- Example: `SUM([Sales]) > SUM([Profit]) * 2`

---

**Top Filter:**
```
Task: Show top 10 products by sales

Steps:
1. Drag "Product Name" to Filters
2. Click "Top" tab
3. Select "By field"
4. Show: Top 10
5. By field: Sales, Sum
6. Click OK
```

**Top Options:**
- Top or Bottom
- Specific number or percentage
- By field (measure)

---

### B. Measure Filters

**Steps:**
1. Drag **Measure** to Filters shelf
2. Choose aggregation (Sum, Average, Min, Max, etc.)
3. Set range or values

**Hands-On Example:**

```
Task: Show only orders with sales between 100 and 1000

Steps:
1. Drag "Sales" to Filters
2. Select aggregation: Sum

4. Minimum: 100
5. Maximum: 1000
```

**4. Special:**

- All values

### D. Context Filters

**What it is:** Filter applied before other filters (creates temporary filtered dataset)

**When to use:**
- Top N filters
- Filters dependent on other filters
- Improve performance with large datasets

**How to create:**
```
Steps:
1. Create regular filter
2. Right-click filter on Filters shelf
3. Select "Add to Context"
4. Filter turns gray (indicates context)
```

**Hands-On Example:**
```
Task: Show top 5 products by sales in Furniture category only

Steps:
1. Drag "Category" to Filters → Select "Furniture"
2. Right-click Category filter → Add to Context (turns gray)
3. Drag "Product Name" to Filters
4. Top tab → Top 5 by Sales
5. Now Top 5 calculated only from Furniture products
```

**Without Context Filter:**
- Top 5 calculated from ALL products first
- Then Category filter applied
- May show fewer than 5 products

**With Context Filter:**
- Category filter applied first
- Then Top 5 calculated from filtered result
- Always shows 5 products (if available)

---

## 6. Sorting

### A. Sort by Toolbar

**Quick Sort:**
1. Click **Sort Ascending** icon (A→Z, 0→9) on toolbar
2. Click **Sort Descending** icon (Z→A, 9→0) on toolbar

**What it sorts:**
- Dimension on Columns or Rows shelf
- Sorts by measure in view

---

### B. Sort Dialog

**Steps:**
1. Click dimension pill on shelf
2. Click **Sort** icon (bar chart with arrow)
3. Sort dialog opens

**Or:**
1. Right-click dimension on shelf
2. Select **Sort**

---

**Sort Options:**

**1. Data source order:**
- Original order from data
- No sorting applied

**2. Alphabetic:**
- Sort by dimension name
- Ascending or Descending

**3. Field:**
- Sort by measure value
- Choose aggregation (Sum, Average, etc.)
- Choose field to sort by
- Ascending or Descending

**4. Manual:**
- Drag items to reorder
- Custom sequence

---

**Hands-On Examples:**

**Sort bars by sales (descending):**
```
Steps:
1. Create bar chart: Sub-Category vs Sales
2. Right-click "Sub-Category" → Sort
3. Sort by: Field
4. Field: Sales, Sum
5. Order: Descending
6. Click OK
```

**Manual sort:**
```
Task: Show regions in specific order (West, East, Central, South)

Steps:
1. Create chart with Region
2. Right-click "Region" → Sort
3. Select "Manual"
4. Drag regions to desired order
5. Click OK
```

**Sort icon indicator:**
- Dimension pill shows sort icon when sorted
- Click to toggle between ascending/descending

## 7. Grouping

**What it is:** Combines dimension members into higher-level categories

**When to use:**
- Simplify data
- Create custom categories
- Combine similar items

---

### Method 1: Visual Grouping (In View)

**Steps:**
1. Create visualization with dimension
2. Select multiple marks (Ctrl+Click or drag to select)
3. Right-click selection → **Group**
4. New group created
5. Appears in Data pane with paper clip icon

**Hands-On Example:**
```
Task: Group states into regions

Steps:
1. Create map with State dimension
2. Select California, Oregon, Washington (West Coast)
3. Right-click → Group
4. Group named "California, Oregon, Washington (group)"
5. Repeat for other regions
```

---

### Method 2: Create Group from Data Pane

**Steps:**
1. Right-click dimension in Data pane
2. Select **Create** → **Group**
3. Group dialog opens
4. Select members to group
5. Click **Group** button
6. Name the group
7. Click OK

**Hands-On Example:**
```
Task: Group sub-categories into meaningful groups

Steps:
1. Right-click "Sub-Category" → Create → Group
2. Select: Bookcases, Chairs, Furnishings, Tables
3. Click "Group" button
4. Rename to "Furniture Items"
5. Select: Binders, Envelopes, Labels, Paper
6. Click "Group" button  
7. Rename to "Office Supplies"
8. Click OK
```

---

**Group Options:**

**Include 'Other':**
- Checkbox in Group dialog
- Creates group for ungrouped members

**Edit Group:**
- Right-click group in Data pane → **Edit Group**
- Add/remove members
- Rename groups

**Ungroup:**
- Select grouped marks in view
- Right-click → **Ungroup**
- Or delete group from Data pane

**Exam Tip:** Groups create new dimension, don't modify original field

## 8. Hierarchies

**What it is:** Drill-down structure for related dimensions

**Default Hierarchies:**
- **Date Hierarchy:** Year → Quarter → Month → Day
- **Geographic Hierarchy:** Country → State → City

---

### Create Custom Hierarchy

**Steps:**
1. Drag dimension onto another dimension in Data pane
2. Name hierarchy
3. Add more levels by dragging dimensions into hierarchy
4. Arrange order (top = highest level)

**Hands-On Example:**
```
Task: Create Product Hierarchy

Steps:
1. In Data pane, drag "Category" onto "Sub-Category"
2. Dialog: "Create hierarchy" → Name it "Product Hierarchy"
3. Drag "Product Name" into "Product Hierarchy"
4. Order: Category → Sub-Category → Product Name
5. Click OK
```

---

**Using Hierarchies in View:**

**Drill Down:**
1. Use hierarchy dimension in view
2. Shows top level by default
3. Click **+** icon on dimension pill to drill down
4. Click **-** icon to drill up

**Or:**
1. Click **+** icon on marks in view
2. Drills down to next level

**Hands-On Example:**
```
Using Product Hierarchy:

1. Drag "Product Hierarchy" to Rows
2. Shows Category level
3. Click + on "Furniture" mark
4. Expands to show Sub-Categories under Furniture
5. Click + on "Chairs"
6. Shows individual products in Chairs
```

---

**Hierarchy Options:**

**Remove from Hierarchy:**
- Right-click field in hierarchy → **Remove from Hierarchy**

**Delete Hierarchy:**
- Right-click hierarchy name → **Remove Hierarchy**
- Fields remain in Data pane

**Reorder Levels:**
- Drag fields within hierarchy to reorder


**Task:** Show monthly sales trend, allow user to select year

```
Steps:
1. Order Date (Month) on Columns
2. Sales on Rows
3. Order Date (Year) to Filters

```

---

**Task:** Show profit by sub-category, highlight losses in red

**Solution:**
```
Steps:
1. Sub-Category on Rows, Profit on Columns
2. Drag "Profit" to Color
3. Click Color → Edit Colors
```

```
1. Category on Rows, Sales on Columns

```

## Worksheet Tips & Tricks

**1. Swap Rows and Columns:**
- Icon in toolbar or Ctrl+W
- Quickly changes orientation

**2. Show/Hide Elements:**
- **Show Mark Labels:** Click T icon on toolbar
- **Show/Hide Titles:** Right-click title area
- **Show/Hide Field Labels:** Analysis → Table Layout

**3. Duplicate Sheet:**
- Right-click sheet tab → Duplicate
- Keeps all formatting and filters

**4. Clear Sheet:**
- Worksheet → Clear → Sheet
- Removes all fields, resets view

**5. Fit Options:**
- **Fit Width:** Entire view fits width
- **Fit Height:** Entire view fits height
- **Entire View:** Fits both dimensions
- **Standard:** Default size
- Located in toolbar dropdown

**6. Undo/Redo:**
- Ctrl+Z (Undo)
- Ctrl+Y (Redo)

**7. Format Pane:**
- Right-click any element → Format
- Opens formatting pane on left
- Format fonts, colors, alignment, borders, shading
